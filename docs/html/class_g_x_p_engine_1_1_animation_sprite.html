Detailed Description

Animated Sprite. Has all the functionality of a regular sprite, but supports multiple animation frames/subimages.

Constructor & Destructor Documentation

int  cols,

)

Initializes a new instance of the GXPEngine.AnimSprite class.

Parameters
filenameThe name of the file to be loaded. Files are cached internally. Texture sizes should be a power of two: 1, 2, 4, 8, 16, 32, 64 etc. The width and height don't need to be the same. If you want to load transparent sprites, use .PNG with transparency.
colsNumber of columns in the animation.
rowsNumber of rows in the animation.
framesOptionally, indicate a number of frames. When left blank, defaults to width*height.
